# Bisaya AI (BAI) Chat App

A localized AI chatbot that replies in Bisaya with cultural flavor and fun personas

## Pages
- **Login Page** – User authentication and app entry  
- **Main Chat Page** – Chat with the AI, switch personas, and use voice features  
- **Random Quotes Page** – Daily Bisaya quote with option to browse past ones  
- **Saved Chats Page** – View and revisit saved conversations  
- **Developers/About Page** – Meet the team behind the app  

## Features
- Bisaya AI chat  
- Persona modes (Conyo, Bai na Bai)  
- Voice support (STT & TTS)  
- Bisaya quote of the day  
- Save chats and favorites  
- Replies only in Bisaya  
- Developer info database
